## Step By Step

1. Cloning the Repo

```sh
    git clone https://github.com/WayanAndika/dimention.git
```
2. install composer

```sh
    composer install
```
3. install node modules

```sh
    npm i
```
4. Copy file .env.example

```sh
    cp .env.example .env
```
5. Membuat key

```sh
    php artisan key:generate
```

6. Jalankan Aplikasi

   Jalankan bersamaan di terminal yang berbeda
```sh
    php artisan serve
```
```sh
    npm run watch
```
